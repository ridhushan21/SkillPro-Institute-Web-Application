# 🎓 SkillPro Institute Web Application

SkillPro Institute is a modern, responsive web application designed to showcase vocational training programs, manage student interactions, and enhance user engagement through an intuitive and interactive platform.

## 🚀 Features

* 🌐 Responsive and user-friendly web design
* 🔐 User Authentication (Login & Registration)
* 📚 Course Listing with detailed information
* 👨‍🏫 Instructor Profiles with expertise details
* 📅 Event Calendar for upcoming programs
* 📩 Contact & Inquiry System
* 🧑‍💼 Admin Dashboard for system management
* 🔎 Search and navigation optimization

## 💡 Project Overview

This web application was developed to digitally represent a vocational training institute, allowing users to explore courses, interact with instructors, and manage their learning journey. The system improves accessibility, engagement, and efficiency compared to traditional static websites. 

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Tools:** VS Code, XAMPP, GitHub

## 🧠 Key Functionalities

* 🔐 Secure login and registration system
* 📊 Dynamic admin dashboard for managing users, courses, and inquiries
* 📚 Course enrollment and management system
* 📩 Inquiry submission and handling
* 📅 Event calendar with upcoming schedules
* ⚙️ Backend integration using PHP & MySQL

## 📂 Project Structure

* `index.php` – Homepage
* `login.php / register.php` – Authentication
* `courses.php` – Course listings
* `instructors.php` – Instructor profiles
* `contact.php` – Inquiry form
* `dashboard.php` – Admin panel
* `config.php` – Database connection

## 🎯 Objective

The main goal of this project is to develop a scalable and user-friendly web platform that enhances institutional visibility, simplifies course management, and improves user interaction through modern web technologies.

## 🌐 Platform

* Web-based (Responsive for Desktop & Mobile)
